# 使用方法
- 根据 `data/README.md` 中的说明，生成支付宝或微信的收支明细 CSV 文件，并存放到 `data/` 目录下
- 在 memory/USER.md 中编辑你的个人信息，然后保存，我自己是从 Claude 聊天里直接复制出来的
- 进入到当前目录下，执行 `claude`
- 然后在 Claude 里执行：
```md
加载你对我之前的记忆 `memory/USER.md`，然后根据这个记忆，继续开始按照 financial-analyzer 的流程，进行深度分析，一定要给我一种 wow experience。
```
